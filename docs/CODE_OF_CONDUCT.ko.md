# 기여자 행동 강령(Contributor Covenant Code of Conduct)

[English](../CODE_OF_CONDUCT.md) | **한국어**

## 우리의 다짐

우리는 구성원, 기여자, 메인테이너로서 나이, 체형, 가시적이거나 비가시적인 장애,
민족, 성적 특징, 성 정체성과 표현, 경험 수준, 학력, 사회경제적 지위, 국적, 외모,
인종, 종교, 성적 정체성과 지향에 관계없이 모든 사람에게 우리 커뮤니티 참여가
괴롭힘 없는 경험이 되도록 만들 것을 다짐합니다.

우리는 개방적이고, 환영하며, 다양하고, 포용적이며, 건강한 커뮤니티에 기여하는
방식으로 행동하고 소통할 것을 다짐합니다.

## 우리의 기준

긍정적인 환경에 기여하는 행동의 예:
- 공감과 친절함을 보이기
- 서로 다른 의견과 경험을 존중하기
- 건설적인 피드백을 주고받으며 너그럽게 수용하기
- 책임을 지고 실수로부터 배우기
- 커뮤니티 전체에 가장 이로운 것에 집중하기

용납되지 않는 행동의 예:
- 성적인 언어, 이미지 또는 관심 표현
- 트롤링, 모욕적이거나 비하하는 발언
- 공개적이거나 사적인 괴롭힘
- 허락 없이 타인의 사적 정보를 공개하기
- 전문적인 환경에서 부적절한 그 밖의 행위

## 시행 책임

프로젝트 메인테이너는 용납되는 행동의 기준을 명확히 하고 시행할 책임이 있으며,
필요한 경우 공정한 시정 조치를 취합니다.

## 적용 범위

이 행동 강령은 모든 프로젝트 공간에 적용되며, 개인이 공개적인 장소에서 공식적으로
프로젝트를 대표하는 경우에도 적용됩니다.

## 시행

용납되지 않는 행동은 저장소의 Discussions나 Issues를 통해(민감하지 않은 사안의
경우) 메인테이너에게 신고해 주십시오. 민감한 신고의 경우, 저장소 프로필에 명시된
채널을 통해 메인테이너에게 비공개로 연락해 주십시오.

모든 신고는 신속하고 공정하게 검토되고 조사됩니다.

## 시행 지침

커뮤니티 영향 지침은 Contributor Covenant에서 채택되었으며 다음을 포함할 수
있습니다:
1. 시정(Correction)
2. 경고(Warning)
3. 일시적 금지(Temporary ban)
4. 영구적 금지(Permanent ban)
